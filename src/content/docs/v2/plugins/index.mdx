## Official Tauri Plugins

- [authenticator](https://github.com/tauri-apps/plugins-workspace/tree/dev/plugins/authenticator): Interface with hardware security keys
- [autostart](https://github.com/tauri-apps/plugins-workspace/tree/dev/plugins/authenticator): Automatically launch your app at system startup
- [fs-extra](https://github.com/tauri-apps/plugins-workspace/tree/dev/plugins/authenticator): File system methods that aren't included in the core API
- [fs-watch](https://github.com/tauri-apps/plugins-workspace/tree/dev/plugins/authenticator): Watch the filesystem for changes
- [localhost](https://github.com/tauri-apps/plugins-workspace/tree/dev/plugins/authenticator): Use a localhost server in production apps
- [log](https://github.com/tauri-apps/plugins-workspace/tree/dev/plugins/authenticator): Configurable logging

You get the point...
